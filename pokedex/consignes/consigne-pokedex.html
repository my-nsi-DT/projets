Premières balises

Copier coller (clique droit "copier", puis dans programiz -> clique droit coller") la ligne suivante :

<h1> Pokedex </h1>

h1 est une balise, cela correspond à un titre : Tout le texte dans la balise sera un titre.

Ecrire en dessous du titre :
dresseur : nom_d_equipe
Ajouter maitenant la balise b : elle permet de mettre en gras du texte.
Le debut de la balise <b> se place devant le texte ciblé, et on ferme la balise avec un / : <b> placé derrière le texte. Exercice : Rendre le texte Dresseur en gras :

<b>Dresseur:</b>

Le saut de ligne, une balise orpheline

Ecrire le texte suivant en dessous du nom de groupe :
Cordée Numérique
Mayotte
Comment faire pour sauter une ligne ? Tester la touche "Entrée".
-> Pour sauter une ligne il faut ajouter la balise <br> , cette balise est dites orpheline, c'est à dire qu'on ne la ferme pas (pas de </br>). Exercice : Ajouter une balise br afin de sauter une ligne

Question 1.3 : hr est également une balise orpheline, mais que fait-elle ?

Balises imbriquées

On peut appliqué plusieurs balises à un texte, par exemple pour centrer un titre (deux solutions):
  • <h1><center>Titre centré</center></h1>
  • <center><h1>Titre centré</h1></center>

Ajout d'images

  1. Télécharger l'image suivante (clique droit, sauvegarder l'image sous)
  2. Charger l'image précédemment téléchargée (Upload Asset)

  3. Ajouter la balise pour avoir une image :<img src="./pokedex.png"/>
