Pikachu le premier Pokémon

Objectif

Le but est de créer notre toute première carte pokemon : Pikachu.
L'objectif est de faire la carte Pokemon ( rappel ici des balises)
Consignes :
  1. Ecrire "Pikachu" en titre principal centré
  2. Ecrire en italique centré"N°25 Pokémon Souris Taille : 0.4m Poids : 6.0 Kg"
  3. Ajouter l'image de Pikachu (image libre)
  4. Ecrire "Attaques" comme un titre secondaire souligné
  5. Ecrire "Caractéristiques" comme un titre secondaire souligné
  6. Ajouter les Caractéristiques
  7. Ajouter deux attaques sous forme de listes


Exemple

Nouveau Fichier

On va pour cela créer un nouveau fichier : New HTML File

Nommer le fichier pikachu

Lien Pokedex

Revenir dans le fichier du pokedex et ajouter un lien : <a href="./pikachu.html">Pikachu</a>

Bonus

Créér une nouvelle carte pour le pokemon de votre choix.
