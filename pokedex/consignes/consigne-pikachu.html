Pikachu le premier Pokémon

Objectif

Le but est de créé notre toute première carte pokemon : Pikachu. On va pour cela créer un nouveau fichier : New HTML File

Nommer le fichier pikachu
L'objectif est de faire la carte Pokemon ( rappel ici des balises) Consignes :
  1. Ecrire "Pikachu" en titre principal centré
  2. Ecrire en italique centré"N°25 Pokémon Souris Taille : 0.4m Poids : 6.0 Kg"
  3. Ajouter l'image de Pikachu (image libre)
  4. Ecrire "Attaques" comme un titre secondaire souligné
  5. Ecrire "Caractéritiques" comme un titre secondaire souligné
  6. Ajouter les Caractéristiques
  7. Ajouter deux attaques sous forme de listes
